# ⚽ Football Player Market Value Prediction

A machine learning project that predicts the market value (in Euros) of professional 
football players based on their FIFA-rated attributes, physical characteristics, and 
performance metrics.

## 📌 Overview

This project applies supervised regression techniques to a dataset of ~17,000 football 
players sourced from SoFIFA.com. It covers the full ML pipeline — from exploratory data 
analysis and preprocessing through to model development, evaluation, and comparison.

## 🎯 Objective

To build and evaluate regression models that can accurately estimate a player's market 
value from attributes such as age, skill ratings, physical stats, and positional data — 
supporting real-world use cases in player recruitment, contract negotiation, and transfer 
market analysis.

## 📦 Dataset

- **Source**: [Football Players Data — Kaggle](https://www.kaggle.com/datasets/maso0dahmed/football-players-data)
- ~17,954 players | 51 features | Mix of numerical and categorical attributes

## 🛠️ Tech Stack

- Matplotlib, Seaborn
- Scikit-learn
